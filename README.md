## firebase config
You must create /src/firebase/firebaseConfig.ts file to make firebase work.

example firebaseConfig.ts:
export default {
  apiKey: 'xxxx',
  authDomain: 'xxxx',
  projectId: 'xxxx',
  storageBucket: 'xxxx',
  messagingSenderId: 'xxxx',
  appId: 'xxxx',
  measurementId: 'xxxx',
};
